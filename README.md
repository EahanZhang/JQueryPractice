# JQueryPractice

AddELement

append()            --在被选元素的结尾插入内容
prepend()           --在被选元素的开头插入内容
after()             --在被选元素之后插入内容
before()            --在被选元素之前插入内容


删除
remove()            --删除被选元素及其子元素
empty()             --删除被选元素的子元素

remove()可以接受一个参数，可以对被删元素进行过滤:
$("#div").remove(".italic");

empty()不接受任何参数.
